Keybinds Cheatsheet

A compact reference for the bindings actually present in ~/.config/nvim.
Leader: Space
Localleader: Space
Theme: Tokyo Night Storm
<leader> means Space
C-x means Ctrl + x
n normal mode
i insert mode
t terminal mode

Core

Space wWrite file
Space qQuit window
Space QForce close buffer
Space nnToggle line numbers
Space yCopy to system clipboard
Space pPaste from system clipboard
Space ?Show buffer-local keymaps

Find

Space ffFind files
Space fgLive grep
Space fbOpen buffers
Space fhHelp tags
C-nReveal current file in Neo-tree
The dashboard also exposes Find File and Recent Files as single-key startup actions.

Panes

Space svVertical split current file
Space ssHorizontal split current file
Space sfFind file in vertical split
Space sFFind file in horizontal split
Space seEqualize split sizes
Space sxClose current split

Move / Resize

C-hMove left
C-jMove down
C-kMove up
C-lMove right
↑ / ↓Resize height
← / →Resize width

Terminal

C-\Toggle terminal
Space thHorizontal terminal
Space 1Terminal 1 · server logs
Space 2Terminal 2 · tests
EscLeave terminal insert mode
jkLeave terminal insert mode
Inside terminals, C-h/j/k/l also moves across splits.

LSP + Code

gd / KDefinition / hover docs
Space rnRename symbol
Space caCode action
Space fFormat current buffer
Space vsSelect Python virtual environment
LSP bindings appear only after a language server attaches.

Buffers

L / HNext / previous visible buffer
Space bn / bNNext / previous buffer
Space bxClose current buffer
Space bXClose other buffers
Space bpPin buffer
Space b1-4Jump to buffer slot 1-4
Bufferline shows open files across the top; panes remain your layout tool.

Completion

C-SpaceOpen menu
Tab / S-TabNext / previous item
EnterConfirm selected
C-eAbort completion

Git

Space gpPreview current Git hunk

Markdown

Space umToggle rendered Markdown
Rendering is conservative: normal / command / terminal modes only, not insert mode.

Dashboard

e / f / rNew file / find file / recent files
s / tOpen server / test terminal
c / u / qConfig / update plugins / quit
Generated from local config files in ~/.config/nvim Tip: press Space and wait for which-key to explore groups interactively.
